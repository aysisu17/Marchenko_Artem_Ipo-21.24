```cs

{
    public int rows;
    public int cols;

    public void Display()
    {
        Console.WriteLine($"Строк: {rows}, Столбцов: {cols}");
    }
}
```

# 2.6. Объявите класс Manager c полями age (целочисленный тип, возраст) и name (строковый тип, имя). Создайте метод GetAge для получения значения поля age и метод GetName для получения значения поля name.

```cs
class Manager
{
    public int age;
    public string name;

    public int GetAge()
    {
        return age;
    }

    public string GetName()
    {
        return name;
    }
}
```
